# Ray Tracing in One Weekend

This project is based on Peter Shirley's Ray Tracing in One Weekend series.

# Dependencies

This project depend on the unit testing framework Catch2.

# How to use it

## Build

    $ mkdir build
    $ cd build
    $ cmake .. -DCMAKE_BUILD_TYPE=Release
    $ cmake --build . -- -j4

## Run unit tests

    # ./tests

## Run benchmarks

    # ./benchmarks

## Run ray tracer

    # ./rtiow
